# web_labs
Опис проєкту:  
Це веб-застосунок для перегляду інформації про популярні фільми та серіали. Користувач може переглядати фільми, шукати їх за назвою, фільтрувати за жанром або роком випуску, а також переходити між сторінками “Головна”, “Фільми”, “Серіали” та сторінками жанрів.
## Інструкція щодо запуску

1. Клонувати репозиторій або завантажити ZIP-файл:
```bash
git clone <URL_репозиторію>
Відкрити папку проєкту.

Відкрити файл index.html у браузері:

Просто двічі клікнути на index.html, або

Відкрити через меню браузера → “Відкрити файл”.

Використовувати веб-застосунок:

Пошук фільмів за назвою у полі пошуку.

Фільтрація за жанрами та роком.

Навігація між сторінками за допомогою кнопок у шапці.
